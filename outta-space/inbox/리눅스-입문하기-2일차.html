리눅스 입문하기 2일차 ​

리눅스 셸과 기본 명령어

명령어 - 셸에 명령어 [옵션] [인자1] [인자2] 형태로 입력

셸의 종류 - sh, csh, ksh, tcsh, bash, zsh, fish (시간 순) &gt 누가 언제 개발했는지, 특징들 기록하기 (리눅스마스터 대비)

명령어 종류

ls - 디렉토리 확인 (옵션: -l, -a)

pwd, whoami, hostnamectl(시스템 정보)

touch - 파일 생성하기

-a: 접근 시간을 변경

-m: 수정 시간을 변경

-t: 위 옵션에 시간 인자 전달, [[CC]YY]MMDDhhmm[.ss]

stat: 파일 정보를 확인하는 명령어

mkdir: 디렉토리를 만듦

rmdir: 디렉토리 삭제

리눅스 파일 시스템

의 다섯가지 특징

  • 대소문자를 구분

  • 긴 파일명 지원

  • 링크 파일을 지원

  • 확장자가 파일 형식을 강제하지 않음 (그냥 표시의 역할만 함)

  • 파일과 디렉토리에 권한이 있음

리눅스 파일의 종류

  • 디렉터리 파일: d

  • 일반 파일: -

  • 장치 파일 (문자 장치, 블록 장치 파일): c, b

  • 링크 파일: l

  • 통신 파일 (소켓 파일, 명명된 파이프 파일): s, p

  • 기본 트리 구조

  • /, /bin, /boot, /dev, /etc, /home, /lib, /media, /mnt, /opt, /proc, /root, /run, /sbin, /srv, /sys, /tmp, /usr, /var

Getconf 명령어 - 시스템에 미리 정의된 제한이나 변수를 확인할 때 사용

  • Getconf [옵션] [변수] [경로]

  • 출력은 바이트 단위

절대 경로와 상대 경로 - 이미 앎

  • Cd, cd ~, .와 .. 같은거

  • Pwd

파일 복사하고 이동하기

  • Mkdir -p를 이용하면 하위 디렉터리까지 생성 가능

  • Tree 명령어는 설치해야 하는 거였음

  • Cp 명령어

  • Mv 명령어 (디렉터리 이동도 되는 구나)
