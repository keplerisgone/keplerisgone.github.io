데이터베이스 ​

무결성 제약조건 ​

  • 유형: 무결성 제약조건, 도메인 무결성, 개체 무결성, 참조 무결성
  • 규칙: 도메인 무결성은 속성 값의 범위·자료형·NULL 허용 여부를 제한하고, 개체 무결성은 기본키가 NULL이 될 수 없도록 하며, 참조 무결성은 외래키가 참조 대상 기본키와 일치하도록 보장한다.
  • 함정: 제약 대상이 속성이면 도메인, 튜플과 기본키가 나오면 개체, 속성과 튜플·외래키가 나오면 참조 무결성이다.
  • 예시: 속성 값 제한 → 도메인 / 기본키 NULL 금지 → 개체 / 외래키 참조 보장 → 참조

네트워크 ​

CRC ​

  • 유형: 오류 검출, CRC, 데이터 링크 계층
  • 규칙: CRC는 Cyclic Redundancy Check의 약자로, 데이터를 다항식 기반으로 나눈 나머지를 검사값으로 붙여 전송하고 수신 측에서 다시 계산해 오류 여부를 확인한다.
  • 함정: CRC는 오류를 검출하는 기법이지 직접 수정하는 기법은 아니다. 3글자 영어 약자, 체크섬, 다항식, 2진수 나눗셈, 나머지 값이 나오면 CRC를 의심한다.
  • 예시: 송신 측이 데이터에 CRC 값을 붙여 보내고, 수신 측 계산 결과가 맞지 않으면 전송 오류로 판단한다.

서브넷 마스크와 블록 크기 ​

  • 유형: 서브넷 마스크, 블록 크기, 브로드캐스트 수신 가능 IP
  • 규칙: 서브넷 마스크가 255.255.252.0이면 세 번째 옥텟 기준으로 계산하며, 블록 크기는 256 - 252 = 4이다. 따라서 세 번째 옥텟은 4개 단위로 묶인다.
  • 함정: 255 - 252 = 3으로 계산하면 안 된다. 서브넷 블록 크기는 255가 아니라 256에서 마스크 값을 빼야 한다.
  • 예시: 192.168.35.10 / 255.255.252.0 → 35는 32~35 구간에 속함 → 네트워크 범위는 192.168.32.0 ~ 192.168.35.255
  • 메모: 255 - 252가 아니라 256 - 252. 3개 단위가 아니라 4개 단위.

Java 예외 처리 ​

java
public class Main {
    public static void main(String[] args) {
        int a = 5, b = 0;

        try {
            System.out.print(a / b);
        } catch (ArithmeticException e) {
            System.out.print("출력1");
        } catch (ArrayIndexOutOfBoundsException e) {
            System.out.print("출력2");
        } catch (NumberFormatException e) {
            System.out.print("출력3");
        } catch (Exception e) {
            System.out.print("출력4");
        } finally {
            System.out.print("출력5");
        }
    }
}
  • 유형: Java 예외 처리, try-catch-finally, ArithmeticException
  • 규칙: try 블록에서 예외가 발생하면 해당 예외 타입과 가장 먼저 일치하는 catch 블록이 실행된다. finally 블록은 예외 발생 여부와 관계없이 마지막에 실행된다.
  • 함정: int에서 5 / 0은 ArithmeticException이 발생한다. 따라서 ArithmeticException의 catch가 실행되고, 이후 finally가 반드시 실행된다.
  • 예시: a = 5, b = 0 → a / b에서 0으로 나누기 예외 발생 → "출력1" 출력 → finally 실행 → "출력5" 출력

c
#include <stdio.h>

char Data[5] = {'B', 'A', 'D', 'E'};
char c;

int main() {
    int i, temp, temp2;

    c = 'C';
    printf("%d\n", Data[3] - Data[1]);

    for (i = 0; i < 5; ++i) {
        if (Data[i] > c)
            break;
    }

    temp = Data[i];
    Data[i] = c;
    i++;

    for (; i < 5; ++i) {
        temp2 = Data[i];
        Data[i] = temp;
        temp = temp2;
    }

    for (i = 0; i < 5; i++) {
        printf("%c", Data[i]);
    }
}
  • 유형: C 문자 배열, 문자 코드 연산, 배열 삽입, 전역 배열 초기화
  • 규칙: C에서 문자는 내부적으로 정수 ASCII 코드로 계산될 수 있다. 문자 배열을 부분 초기화하면 남은 칸은 0, 즉 '\0'으로 초기화된다.
  • 함정: Data[3] - Data[1]은 문자끼리 빼는 것이 아니라 ASCII 코드 차이를 계산한다. 'E' - 'A' = 69 - 65 = 4다. Data[5]는 문자열 리터럴이 아니라 문자 배열이며, 마지막 칸 Data[4]는 자동으로 '\0'이 된다.
  • 예시: {'B','A','D','E','\0'}에 'C'를 정렬 위치에 끼워 넣으면 {'B','A','C','D','E'}가 된다.

C 동적 2차원 배열 ​

c
#include <stdio.h>
#include <stdlib.h>

void set(int** arr, int* data, int rows, int cols) {
    for (int i = 0; i < rows * cols; ++i) {
        arr[((i + 1) / rows) % rows][(i + 1) % cols] = data[i];
    }
}

int main() {
    int rows = 3, cols = 3, sum = 0;
    int data[] = {5, 2, 7, 4, 1, 8, 3, 6, 9};
    int** arr;

    arr = (int**) malloc(sizeof(int*) * rows);
    for (int i = 0; i < cols; i++) {
        arr[i] = (int*) malloc(sizeof(int) * cols);
    }

    set(arr, data, rows, cols);

    for (int i = 0; i < rows * cols; i++) {
        sum += arr[i / rows][i % cols] * (i % 2 == 0 ? 1 : -1);
    }

    for (int i = 0; i < rows; i++) {
        free(arr[i]);
    }
    free(arr);

    printf("%d", sum);
}
  • 유형: C 동적 2차원 배열, 포인터 배열, 인덱스 변환, 삼항 연산자
  • 규칙: set() 함수에서 data[i]를 arr[((i + 1) / rows) % rows][(i + 1) % cols] 위치에 저장하므로, data의 순서가 그대로 3×3 배열에 들어가지 않는다. 이후 sum 계산에서는 arr[i / rows][i % cols]을 row-major 순서로 읽고, i가 짝수면 더하고 홀수면 뺀다.
  • 함정: data = {5,2,7,4,1,8,3,6,9}가 3×3 배열에 순서대로 들어가는 것이 아니다. (i + 1)을 기준으로 행과 열을 계산하므로 한 칸 밀린 듯한 순서가 된다.
  • 예시: 최종 arr은 [[9,5,2],[7,4,1],[8,3,6]]이고, 합은 9 - 5 + 2 - 7 + 4 - 1 + 8 - 3 + 6 = 13이다.

Java 생성자와 오버라이딩 ​

java
public class Main {
    public static void main(String[] args) {
        new Child();
        System.out.println(Parent.total);
    }
}

class Parent {
    static int total = 0;
    int v = 1;

    public Parent() {
        total += (++v);
        show();
    }

    public void show() {
        total += total;
    }
}

class Child extends Parent {
    int v = 10;

    public Child() {
        v += 2;
        total += v++;
        show();
    }

    @Override
    public void show() {
        total += total * 2;
    }
}
  • 유형: Java 상속, 생성자 실행 순서, 오버라이딩, 동적 바인딩, static 변수
  • 규칙: 자식 객체를 생성하면 부모 생성자가 먼저 실행된 뒤 자식 생성자가 실행된다. 단, 부모 생성자 안에서 오버라이딩된 메서드를 호출하면 실행 시점의 실제 객체 타입인 자식 클래스의 메서드가 호출된다.
  • 함정: Parent 생성자 안의 show()는 Parent.show()가 아니라 Child.show()가 실행된다. 또한 Parent의 v와 Child의 v는 서로 다른 필드다. static total은 Parent와 Child가 공유한다.
  • 예시: new Child() → Parent 생성자 실행 → Child.show() 실행 → Child 필드 초기화 → Child 생성자 실행 → Child.show() 실행 → Parent.total 출력
  • 메모: 부모 생성자가 먼저 실행된 후 자식 생성자가 실행된다.

Java 재귀와 Math.max ​

java
public class Main {
    public static void main(String[] args) {
        int[] data = {3, 5, 8, 12, 17};
        System.out.println(func(data, 0, data.length - 1));
    }

    static int func(int[] a, int st, int end) {
        if (st >= end) return 0;
        int mid = (st + end) / 2;
        return a[mid] + Math.max(func(a, st, mid), func(a, mid + 1, end));
    }
}
  • 유형: Java 재귀 함수, 배열, 분할 계산, Math.max
  • 규칙: func(a, st, end)는 구간의 가운데 값 a[mid]를 더하고, 왼쪽 재귀 결과와 오른쪽 재귀 결과 중 더 큰 값을 선택한다.
  • 함정: 모든 값을 더하는 함수가 아니다. 각 단계마다 왼쪽/오른쪽 중 큰 쪽만 선택한다. st >= end이면 0을 반환한다.
  • 예시: [3,5,8,12,17]에서 가운데 8을 더하고, 왼쪽 결과 8과 오른쪽 결과 12 중 큰 12를 더하므로 20이다.

Python 트리 재귀 ​

python
class Node:
    def __init__(self, value):
        self.value = value
        self.children = []

def tree(li):
    nodes = [Node(i) for i in li]
    for i in range(1, len(li)):
        nodes[(i - 1) // 2].children.append(nodes[i])
    return nodes[0]

def calc(node, level=0):
    if node is None:
        return 0
    return (node.value if level % 2 == 1 else 0) + sum(calc(n, level + 1) for n in node.children)

li = [3, 5, 8, 12, 15, 18, 21]

root = tree(li)

print(calc(root))
  • 유형: Python 클래스, 트리 구조, 완전 이진 트리, 재귀 순회
  • 규칙: nodes[(i - 1) // 2].children.append(nodes[i])는 배열 인덱스를 이용해 부모-자식 관계를 만든다. calc()는 level이 홀수인 노드의 value만 더한다.
  • 함정: 리스트 값을 단순히 순서대로 더하는 문제가 아니다. 리스트 인덱스를 완전 이진 트리 구조로 바꾼 뒤 홀수 레벨 노드만 합산한다.
  • 예시: [3,5,8,12,15,18,21] → 트리에서 level 1의 노드는 5, 8 → 5 + 8 = 13
  • 메모: 트리 구조 먼저 그리자. 루트는 level 0이다.

C 연결 리스트 재연결 ​

c
#include <stdio.h>
#include <stdlib.h>

typedef struct Data {
    int value;
    struct Data *next;
} Data;

Data* insert(Data* head, int value) {
    Data* new_node = (Data*)malloc(sizeof(Data));
    new_node->value = value;
    new_node->next = head;
    return new_node;
}

Data* reconnect(Data* head, int value) {
    if (head == NULL || head->value == value) return head;
    Data *prev = NULL, *curr = head;
    while (curr != NULL && curr->value != value) {
        prev = curr;
        curr = curr->next;
    }

    if (curr != NULL && prev != NULL) {
        prev->next = curr->next;
        curr->next = head;
        head = curr;
    }
    return head;
}

int main() {
    Data *head = NULL, *curr;
    for (int i = 1; i <= 5; i++)
        head = insert(head, i);

    head = reconnect(head, 3);

    for (curr = head; curr != NULL; curr = curr->next)
        printf("%d", curr->value);

    return 0;
}
  • 유형: C 구조체 포인터, 단일 연결 리스트, 동적 할당, 노드 재연결
  • 규칙: insert()는 새 노드를 리스트의 맨 앞에 삽입한다. reconnect()는 특정 value를 가진 노드를 찾아 기존 위치에서 떼어낸 뒤 head 앞으로 이동시킨다.
  • 함정: 1, 2, 3, 4, 5 순서로 insert해도 앞에 삽입되므로 리스트는 5→4→3→2→1이 된다. reconnect(head, 3) 이후에는 3이 맨 앞으로 이동해 3→5→4→2→1이 된다.
  • 예시: prev는 curr의 이전 노드를 기억하는 포인터이며, prev->next = curr->next로 curr 노드를 기존 위치에서 제거한다.

C 구조체 배열과 비트 AND ​

c
#include <stdio.h>

typedef struct student {
    char* name;
    int score[3];
} Student;

int dec(int enc) {
    return enc & 0xA5;
}

int sum(Student* p) {
    return dec(p->score[0]) + dec(p->score[1]) + dec(p->score[2]);
}

int main() {
    Student s[2] = { "Kim", {0xA0, 0xA5, 0xDB}, "Lee", {0xA0, 0xED, 0x81} };
    Student* p = s;
    int result = 0;

    for (int i = 0; i < 2; i++) {
        result += sum(&s[i]);
    }

    printf("%d", result);
    return 0;
}
  • 유형: C 구조체 배열, 구조체 초기화, 배열 멤버 초기화, 비트 AND
  • 규칙: 구조체는 멤버가 선언된 순서대로 초기화된다. 구조체 배열도 각 원소의 멤버를 순서대로 채우며, 내부 배열 멤버는 중괄호로 묶어 초기화할 수 있다.
  • 함정: Student s[2] = { "Kim", { ... }, "Lee", { ... } }처럼 바깥 중괄호가 덜 명확해 보여도, C는 Student의 멤버 순서에 맞춰 값을 채운다. name에는 문자열 주소가 들어가고, score[3]에는 3개의 정수가 들어간다.
  • 예시: {"Kim", {0xA0, 0xA5, 0xDB}}는 s[0].name = "Kim", s[0].score = {0xA0,0xA5,0xDB}로 초기화된다. enc & 0xA5를 적용해 두 학생 점수를 합산하면 908이다.

Java 오버로딩과 재귀 ​

java
public class Main {
  public static void main(String[] args) {
    System.out.println(calc("5"));
  }
 
  static int calc(int value) {
    if (value <= 1) return value;
    return calc(value - 1) + calc(value - 2);
  }
 
  static int calc(String str) {
    int value = Integer.valueOf(str);
    if (value <= 1) return value;
    return calc(value - 1) + calc(value - 3);
  }
}
  • 유형: Java 메서드 오버로딩, 재귀 함수, 문자열에서 정수 변환
  • 규칙: calc("5")는 인자가 String이므로 calc(String str)이 먼저 호출된다. calc(String) 내부에서 값은 int로 변환되고, 이후 calc(value - 1), calc(value - 3)은 int 인자를 넘기므로 calc(int value)가 호출된다.
  • 함정: 처음만 calc(String)이고, 재귀 호출은 calc(int)로 넘어간다. calc(String)이 계속 재귀되는 것이 아니다.
  • 예시: calc("5") → calc(String) → value = 5 → calc(4) + calc(2) → calc(int) 기준 계산 → 3 + 1 = 4
