[프로그래밍] 유형: Java 조건문, 논리 연산자, 비교 연산자, 연산자 우선순위 규칙: Java 연산자 우선순위는 ! → 비교연산자(==, >, !=) → & → ^ → | 순서로 계산된다. 함정: ^는 논리 XOR이므로 양쪽 조건이 서로 다를 때만 true이고, &와 |는 단축 평가 없이 양쪽 조건을 모두 계산한다. 예시: 첫 if는 참이므로 w = x + y = 7, 내부 if도 참이 되어 System.out.println(w)로 7 출력

[Java 연산자 우선순위] 유형: 조건식 계산 순서 규칙: 괄호 → 단항 ! → 산술 → 비교 ==, !=, >, < → 논리 AND & → 논리 XOR ^ → 논리 OR | 순서로 계산한다. 함정: ^는 제곱이 아니라 XOR이며, 조건식에서는 양쪽 결과가 다를 때 true가 된다. 예시: false ^ true = true, true ^ true = false

[프로그래밍] 유형: Python 반복문, range, 비트 시프트 연산자, 변수 덮어쓰기 규칙: range(1, 3)은 1, 2까지만 반복하고, a >> i는 a를 오른쪽으로 i비트 시프트하여 2^i로 나눈 몫과 같다. 함정: result = a >> i가 반복마다 기존 result를 누적하지 않고 새 값으로 덮어쓰므로, 마지막 반복 i=2의 결과만 최종 출력에 남는다. 예시: i=1 → 100 >> 1 = 50 → result=51, i=2 → 100 >> 2 = 25 → result=26, 최종 출력 26

[프로그래밍] 유형: Python 2차원 리스트, 인덱싱, 중첩 반복문, print(end=' ') 규칙: 리스트 인덱스는 0부터 시작하며, lol[2][1]은 세 번째 리스트 [6,7,8,9]의 두 번째 값인 7을 의미한다. 함정: print(item, end=' ')는 줄바꿈 없이 공백을 붙여 출력하고, 안쪽 반복문이 끝난 뒤 print()가 실행되어 다음 줄로 넘어간다. 예시: print(lol[0]) → [1, 2, 3], print(lol[2][1]) → 7, 반복문 출력 → 1 2 3 / 4 5 / 6 7 8 9

[프로그래밍] 유형: Java 반복문, 10진수→2진수 변환, 나머지 연산 규칙: n > 0인 동안 n % 2를 배열에 저장하고 n /= 2로 몫을 줄이면 2진수의 비트가 역순으로 저장된다. 함정: 이 코드에서는 n != 0도 결과는 같지만, 양의 정수를 2진수로 변환하는 표준 조건은 n > 0이다. 예시: n=10 → 나머지 0,1,0,1 저장 → 배열을 뒤에서부터 출력하면 00001010

[프로그래밍] 유형: Java while문, continue, 나머지 연산, 누적합 규칙: a++ 후 a%2==1이면 continue로 sum += a를 건너뛰므로 짝수 2,4,6,8,10만 누적된다. 함정: continue를 만나면 반복문이 끝나는 것이 아니라 현재 반복의 남은 코드만 건너뛰고 다음 반복으로 넘어간다. 예시: sum = 2 + 4 + 6 + 8 + 10 = 30 메모: 침착하게 풀자.

[프로그래밍] 유형: C언어 문자열 포인터, 널 문자('\0'), 사용자 정의 길이 함수 규칙: 문자열은 끝에 '\0'이 있으며, while(*p != '\0') 동안 p++와 r++을 반복하면 문자열 길이를 구할 수 있다. 함정: 문자열 "2022"의 길이는 4, "202207"의 길이는 6이며, 마지막 널 문자 '\0'은 길이에 포함하지 않는다. 예시: len("2022") = 4, len("202207") = 6 → a + b = 10
